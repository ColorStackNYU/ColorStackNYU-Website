# How to Contribute

Want to add a resource? It takes 5 minutes and no coding experience needed.

---

## Steps

1. Go to [`public/resources.json`](https://github.com/ColorStackNYU/ColorStackNYU-Website/blob/main/public/resources.json)
2. Click the **pencil icon** (top right) to edit
3. Find the `"resources": [` line and add your resource before the closing `]`
4. Use this format:

```json
{
  "id": "my-resource",
  "title": "Resource Name",
  "description": "What is this resource and why is it helpful?",
  "link": "https://example.com",
  "category": "Interview Prep",
  "dateAdded": "2025-11-04",
  "contributedBy": "Your Name"
}
```

## Categories

- Interview Prep
- Coursework
- Career Growth
- Web Development
- Mobile Development
- Data Science
- System Design

## That's it!

Scroll down and click **"Commit changes"** → select **"Create a new branch"** → click **"Propose changes"**.

We'll review and add it to the site!

---

**Questions?** Open an issue on GitHub or reach out to ColorStackNYU.
